Поля ввода Для инпутов может быть задана маска. Ошибки ввода. Типы инпутов: * текстовый * числовой (counter) * выбор цвета Состояния * hover (не всегда используется) * focused * disabled * заполнено правильно (зеленый) * заполнено неправильно (красный) Нужно также учитывать * текст placeholder-а (в обычном состоянии и в фокусе) * добавление иконки Подвиды инпутов: * выбор времени * выбор даты (календарь) * ввод данных карты * выбор цвета * ввод телефона * выбор файла для загрузки Нужно также задавать правильные типы клавиатуры. Например поля где только числа или номер телефона.
