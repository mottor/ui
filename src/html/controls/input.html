Text fields

Базовый класс поля - mtr-input.
Минимально для инпута нужно назначать этот класс и класс размера. 

<input class="mtr-input mtr-input--m" type="text"/>

Ширина текстового поля по умолчанию = .
Но его можно сделать на всю ширину блока с помощью класса mtr-input--block.

Состояния

Normal
Hover mtr-input--hovered
Focus mtr-input--focused
Disabled mtr-input--disabled

Размеры полей

Small mtr-input--s
Medium mtr-input--m
Large mtr-input--l

Заметки

  1. Для инпутов может быть задана маска.
  2. Ошибки ввода.
  3. Иконки внутри (перед и после ввода текста)
  4. Крестик очистки
  5. текст placeholder-а (в обычном состоянии и в фокусе)

Типы инпутов:

  • текстовый
  • числовой (counter)
  • выбор цвета

Состояния

  • hover (не всегда используется)
  • focused
  • disabled
  • заполнено правильно (зеленый)
  • заполнено неправильно (красный)

Размеры

  • S
  • M
  • L

Подвиды инпутов:

  • выбор времени
  • выбор даты (календарь)
  • ввод данных карты
  • выбор цвета
  • ввод телефона
  • ввод email
  • выбор файла для загрузки

Нужно также задавать правильные типы клавиатуры. Наример, поля где только числа или номер телефона.
